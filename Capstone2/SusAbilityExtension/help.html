SusAbility Help

SusAbility is a browser extension that helps you review an article’s political placement and reliability using source baselines, linked-source patterns, and article-level signals.

What the extension does

  • Analyzes the current page
  • Estimates political placement on the x-axis
  • Estimates reliability/news value on the y-axis
  • Shows which signals influenced the result
  • Lets you open a dashboard for a larger visual view

What the axes mean

X-axis: Shows political placement in these categories: Left, Lean Left, Center, Lean Right, and Right.
Y-axis: Shows reliability / news value. Higher values suggest stronger sourcing, stronger evidence, and better support from known rated sources.

Research background

SusAbility is inspired by academic work on media bias and article-level analysis. Parts of the project draw on ideas from the University of Pennsylvania’s Computational Social Science Lab and the Penn Media Accountability Project (PennMAP).

These references help explain the research direction behind article-level media analysis, topic selection, framing, and political-lean measurement.

About the source database

SusAbility uses a local source baseline database to help estimate political placement and reliability. The extension compares the current article and its outbound links against rated source patterns, then blends that with article-level signals such as semantics, framing, and rhetoric.

What the main signals mean

  • Political starting point: the article’s source baseline from the local ratings database, if matched.
  • Final political placement: the final x-axis position after adjusting for linked sources, semantics, and framing.
  • Known linked sources: outbound links that matched a rated source in the database.
  • Unknown linked sources: outbound links that did not match a rated source.
  • Linked source pattern: the average political lean of the article’s rated outbound links.
  • Reliability score: article-level reliability blended with the source’s baseline reliability.

How the analysis works

  • The extension first tries to match the current source to a known source in the local database.
  • If the source is known, that gives the article a starting political and reliability baseline.
  • The extension then reviews outbound links and compares them to other rated sources.
  • It also checks article-level cues such as wording, issue framing, and narrative framing.
  • The final position is a blend of source baseline the local database, linked-source evidence, and article-level signals.

Buttons

  • Analyze Page: runs the current article analysis.
  • Citation: creates a citation using article metadata.
  • Dashboard: opens the larger chart and explanation view.
  • Settings: opens dashboard settings such as API usage and reference display.

Important note

SusAbility is meant to support critical reading. It does not replace human judgment, and scores should be read as guidance rather than a final verdict.
