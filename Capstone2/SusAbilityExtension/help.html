SusAbility Help

SusAbility is a browser extension that helps you review an article’s political placement and reliability using source baselines, linked-source patterns, and article-level signals.

What the extension does

  • Analyzes the current page
  • Estimates political placement on the x-axis
  • Estimates reliability/news value on the y-axis
  • Shows which signals influenced the result
  • Lets you open a dashboard for a larger visual view
  • Lets you generate APA and MLA citations

What the axes mean

X-axis: Shows political placement in these categories: Left, Lean Left, Center, Lean Right, and Right.
Y-axis: Shows reliability / news value. Higher values suggest stronger sourcing, stronger evidence, and better support from known rated sources.
Overall score: The popup and dashboard also show a simpler 0–100 score for users. This is a user-facing summary score. The y-axis still keeps the extension’s 0–64 reliability scale.

Research background

SusAbility is inspired by academic work on media bias and article-level analysis. Parts of the project draw on ideas from the University of Pennsylvania’s Computational Social Science Lab and the Penn Media Accountability Project (PennMAP).

About the source database

SusAbility uses a local source baseline database to help estimate political placement and reliability. The extension compares the current article and its outbound links against rated source patterns, then blends that with article-level signals such as semantics, framing, and rhetoric.

What the main signals mean

  • Political starting point: the article’s source baseline from the local ratings database, if matched.
  • Final political placement: the final x-axis position after adjusting for linked sources, semantics, and framing.
  • Linked-source pattern: the overall pull from linked sources that were kept in the analysis.
  • Reliability score: article-level reliability blended with the source’s baseline reliability.
  • Topic cues: issue-related wording that can gently shift placement.
  • Story angle: how the article presents the issue overall.
  • Tone and wording: emotionally loaded, subjective, or intense wording that can affect the reading of the article.

How the analysis works

  • The extension first tries to match the current source to a known source in the local database.
  • If the source is known, that gives the article a starting political and reliability baseline.
  • The extension then reviews outbound links and compares them to other rated sources.
  • It also checks article-level cues such as wording, issue framing, and narrative framing.
  • The final result is a blend of source baseline, linked-source evidence, and article-level signals.

Popup progress steps

  • Reading page content: pulls visible page details from the current article.
  • Checking source database: looks for a local source baseline match.
  • Reviewing linked sources: checks outbound links and filters out low-value or non-editorial links.
  • Filling missing metadata: uses API fallbacks only when needed.
  • Scoring results: combines the gathered signals into the final result.
  • Saving results: stores the latest analysis so the popup and dashboard stay in sync.

Dashboard source details

The dashboard includes a Show source details button under the chart. This opens two grouped lists:

  • Sources that influenced placement: linked sources that contributed to the source-pattern analysis.
  • Sources ignored: links that were found but skipped because they were same-site, same-family, social, support, login, or other non-editorial links.

Repeated domains are grouped together so sources like doi.org are not shown as several separate rows.

Buttons

  • Analyze Page: runs the current article analysis.
  • Generate Citation: opens a separate citation view for APA and MLA output.
  • Copy APA / Copy MLA: copies one citation format at a time.
  • Dashboard: opens the larger dashboard view.
  • Settings: opens dashboard settings.
  • Help: opens this help page.

Important note

SusAbility should be used as a guide for review and comparison. Scores and placements are explainable estimates, not absolute truth.
